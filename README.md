## Network Packet Analyzer

## Task 05 – Packet Sniffing Tool
## Description
This project is a Python-based network packet analyzer that captures and analyzes network packets. The tool displays important packet details such as source IP address, destination IP address, protocol type, and payload data.

## Features
. Captures live network packets
. Displays source and destination IP addresses
. Identifies protocol information
. Displays packet payload data
. Command-line interface

## Technologies Used
* Python
* Scapy library

## Requirements
Install Scapy before running the program:

pip install scapy

## How to Run
Run VS Code or terminal as Administrator.

python packet_analyzer.py
The program will start capturing network packets.
Press CTRL + C to stop capturing.
